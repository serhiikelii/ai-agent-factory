Button Pulse Animation

Демонстрация CSS pulse эффекта для кнопок

Базовый pulse эффект

Добавьте класс .button-pulse к любой кнопке для применения анимации. Эффект работает автоматически и повторяется бесконечно.

Различные стили кнопок

Pulse анимация работает с любыми стилями кнопок - градиентами, плоскими цветами, outline и т.д.

Сравнение с обычной кнопкой

Сравните поведение кнопки без анимации и с pulse эффектом.

Как использовать

Шаг 1: Подключите CSS файл в <head>:
<link rel="stylesheet" href="button-pulse.css">

Шаг 2: Добавьте класс к кнопке:
<button class="button-pulse">Click Me</button>

Accessibility: Анимация автоматически отключается для пользователей с prefers-reduced-motion настройкой в системе.
